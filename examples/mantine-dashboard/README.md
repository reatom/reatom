# Reatom 1001 RC + Mantine 9.2 dashboard

Standalone Vite example that demonstrates idiomatic Reatom app structure with Mantine UI.

```bash
cd examples/mantine-dashboard
npm install
npm run dev
```

Demo credentials:

- email: `alex@example.com`
- password: `password`

Showcases public/private routing, route loaders, forms, actions, effects, async fetches, persistence, fake backend delays, URL search codecs, and optimistic rollback.
